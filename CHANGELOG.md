# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-15

### Added
- Initial release of AI-Powered SQL Agent
- LCEL-based architecture for composable AI chains
- Natural Language Query (NLQ) to SQL conversion
- IBM Granite 3.2 8B Instruct LLM integration
- MySQL database support
- Error recovery and handling mechanisms
- Interactive command-line interface
- Comprehensive documentation
- Configuration management via environment variables
- Type hints throughout codebase
- Support for complex multi-table queries
- Token-efficient schema retrieval

### Features
- `SQLAgent` class for easy initialization
- `SQLAgentConfig` for flexible configuration
- `LLMConfig` for LLM parameter tuning
- Interactive mode for exploratory queries
- Batch query processing support
- Verbose logging for debugging

### Documentation
- README with quick start guide
- ARCHITECTURE.md explaining LCEL design
- EXAMPLES.md with usage examples
- CONTRIBUTING.md for contributors
- API documentation in docstrings

### Testing
- Basic test structure
- Configuration validation tests
